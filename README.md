**Description**

### this is a front end project, 3d software, where you can modify mesh, interact with 3d text and more. ###

# Technologies used: #
- React three fiber
- tailwind.css
- react-use
- use-react-screenshot
- framer-motion
- material UI

  # usage #
  - You can choose color, metalness and roughness for the mesh and plane object.
